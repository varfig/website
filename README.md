# VARFIG — webpage

Landing estática de VARFIG. HTML/CSS/JS plano, sin build, servida por GitHub Pages.

- `index.html` — sitio completo (estilos inline + JS vanilla: reloj UTC, situación portuaria simulada, narrativa sticky por scroll).
- `.nojekyll` — desactiva el procesamiento Jekyll de GitHub Pages.

Diseño origen: proyecto Claude Design *"Radical anti-marketing for VARFIG"*.

## Publicar
Pages sirve desde `main` / raíz. Cualquier push a `main` actualiza el sitio.
